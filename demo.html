Note value recognition for polyphonic music

This is the accompanying web page for the following paper:
  • Eita Nakamura, Kazuyoshi Yoshii, Simon Dixon
    Note Value Recognition for Rhythm Transcription Using a Markov Random Field Model for Musical Scores and Performances of Piano Music
    Manuscript submitted to IEEE/ACM Transactions on Audio, Speech and Language Processing [arXiv:1703.08144]

A rhythm transcription algorithm converts a MIDI performance to a music score representation. Our main contribution is the development of a method that can recognise the score-written quantised note lengths from an input MIDI performance and a result of onset quantisation. This serves as a postprocessing to existing onset rhythm transcription methods, which together can produce complete score information with onset and offset score times.

Source code

You can download source code here. An onset rhythm transcription method based on metrical HMM and an evaluation tool are included.
